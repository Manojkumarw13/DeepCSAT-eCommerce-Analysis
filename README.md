# DeepCSAT - eCommerce Customer Support Analysis

This repository contains the analysis for the DeepCSAT project, focusing on eCommerce Customer Support Data.

## Organization

- `eCommerce_Customer_support_data.csv`: The core dataset containing customer support interactions.
- `DeepCSAT – Ecommerce.pptx`: Presentation outlining the DeepCSAT methodology or findings.
- `Sample_ML_Submission_Template-2.ipynb`: Jupyter Notebook containing the data exploration, preprocessing, visualization, and modeling steps.
